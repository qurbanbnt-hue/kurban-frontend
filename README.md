# Kurban Digital Frontend

Platform terpadu untuk mengelola dan memantau proses penyembelihan hewan qurban secara real-time oleh Yayasan Bhakti Nusa Tenggara.

## Setup Development

1. Install Node.js dari https://nodejs.org/
2. Install dependencies: `npm install`
3. Jalankan development server: `npm run dev`

## Deployment

Project ini dideploy menggunakan Vercel dengan konfigurasi di `vercel.json`.

## Struktur Project

- `index.html` - Halaman login utama
- `admin/` - Halaman admin
- `user/` - Halaman user
- `api/proxy.js` - API proxy untuk Vercel
- `assets/` - CSS, JS, dan gambar
